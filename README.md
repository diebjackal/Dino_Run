# Unity Dino Run Game
볼트를 이용해 간단한 공룡 게임을 만들었습니다

# 사용 중인 유니티 버전
 2019.4.19f1

# 기능

1. 랜던 장애물 생성기능이 있습니다.
2. 점수가 증가하면 공룡의 이동속도가 증가합니다.
3. 이동 거리에 따라 점수가 증가합니다.
4. 최고 점수를 기록하고 볼 수 있습니다.
